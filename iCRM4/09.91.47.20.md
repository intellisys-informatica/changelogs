:arrow_up: Nova categoria 57 - Faturas cobrança parcela - e-mail em Mensagem Template

Foi acrescentada na `Mensagem template` a categoria `57 - Faturas cobrança parcela - e-mail` no projeto ICRM4Utils. A tela de `Cobrança Administradora` foi atualizada para ler essa nova categoria no campo `Template Para E-mail das Parcelas`.

:warning: Correção no envio de parcela por email pela tela de parcela Cobrança

O envio de parcela por email não estava funcionando quando solicitado pela tela de parcela Cobrança. Foi criada a função necessária para realizar o envio corretamente.

:star: Envio de parcela por email dentro do Evento 18 - Arquivo Cobrança Banco

Foi adicionado o envio de parcela por email dentro do `Evento do tipo 18 - Arquivo Cobrança Banco`. Caso o cliente tenha mais de uma parcela, será enviado um anexo contendo todas as parcelas que foram enviadas para o banco.

:star: Novo evento 32 - Parcelas de Cobranças a Vencer

Foi criado um novo evento `32 - Parcelas de Cobranças a Vencer` para enviar automaticamente as parcelas de cobrança que estão próximas do vencimento.

Os critérios para o boleto ser enviado são:

1. Faltando X dias para o vencimento (esse valor deve ser configurado no evento)
2. A parcela deve estar no status de **Em Aberto**
3. A cobrança deve estar no status de **Negociada**
